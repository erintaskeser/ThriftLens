# ThriftLens iPhone / GitHub Pages sürümü

Bu paket GitHub Pages'e yüklenmek için hazırdır.

## GitHub Pages'e yükleme

1. GitHub'da `thriftlens` isimli Public repository oluştur.
2. Bu ZIP'i aç.
3. İçindeki bütün dosyaları repository'nin ana sayfasına yükle.
4. GitHub'da Settings > Pages bölümüne gir.
5. Source: Deploy from a branch
6. Branch: main, Folder: /root
7. Save.

Site adresin yaklaşık şöyle olur:
https://KULLANICI-ADIN.github.io/thriftlens/

## iPhone'a uygulama gibi ekleme

1. Siteyi Safari'de aç.
2. Paylaş simgesine dokun.
3. Ana Ekrana Ekle seçeneğine bas.
4. Add / Ekle de.

Önemli: ChatGPT dosya önizlemesinde değil, GitHub Pages linkinde açılmalıdır.
