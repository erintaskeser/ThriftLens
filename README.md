# ThriftLens V2

Bu sürüm GitHub Pages için hazırdır.

## Yenilikler
- Aynı anda 12 fotoğrafa kadar yükleme
- Fotoğraf galerisi ve tek tek silme
- Tam ürün adı alanı
- Marka, tahmini dönem ve ürün türü
- Vintage ve yeniden satış puanı
- Kanıtlar
- AL / BIRAK önerisi
- eBay, Depop ve Facebook Marketplace arama bağlantıları

## GitHub'a güncelleme
Mevcut ThriftLens repository'ndeki eski dosyaların üzerine bu paketteki dosyaları yükleyin.
GitHub Pages ayarı aynı kalabilir.

Not: Bu sürüm gelişmiş arayüz prototipidir. Gerçek otomatik görsel tanıma ve canlı fiyat verisi için backend/API gerekir.
